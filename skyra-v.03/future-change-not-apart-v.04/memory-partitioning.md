# Memory Partitioning

Memory is partitioned by relationship within the being.

Each relationship a being has gets its own memory partition. Experience lived inside the prefrontal/strategy relationship lives in that partition. Experience lived inside the prefrontal/values relationship lives in a different partition.

But a being can search across any partition whenever it wants. The partitions are organizational, not walls. When prefrontal needs to reason about something, it can pull from any relationship's memory — not just the one currently active.

The partitioning gives retrieval structure. The cross-partition search gives the being its full experiential range.
