# Relationship Lifecycle v0

## Status

This document is partially retired.

The formal language-table model described in earlier `v.03` drafts is no longer
canon.

Retired from this file:

- shared or centrally persisted language records
- `current_language` slots
- stored `forming`, `callable`, and `retired` status fields
- kernel-created language table entries as proof that language stabilized

Still live:

- genome-seeded relationships exist earlier on the creation path
- earned peer relationships begin when previously unconnected beings first meet
- both beings already carry base language from their creation path
- base language is intrinsic to the being and not relationship-owned
- callable language is recognized through cognition and retained experience,
  not by a formal table field

A rewritten lifecycle document should replace this file with the distributed
discovery, local exchange-record, and edge-creation model.
