# Modeling Interaction

This document locks the current top-level interaction primitive.

## `act`

`act` is the top-level primitive.

Definition:

- `act = any intentional boundary engagement that produces an effect or signal`

Final shape:

```text
act:
  target     # who / what it's directed at
  content    # what is being conveyed
  modality   # how it's delivered
  timestamp  # when is it going to happen
  reason     # why it was done
```
