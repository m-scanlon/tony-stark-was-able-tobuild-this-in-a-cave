# Boundary Beings v0

## Purpose

This document locks the world-boundary model in the current `v.03` canon.

## Core Framing

Boundary beings are where the runtime meets the outside world.

Some are singleton genome beings.

Others are created later from genome templates when a being's purpose requires
them.

## The Sensory Layer

The sensory layer has three tiers.

### 1. Peripheral Input Beings

These are specialized beings that face the outside world.

They are purpose-bound and created when needed.

### 2. Main Sensory Being

The main sensory being is the aggregation point.

It is a singleton genome being.

It receives from peripheral input beings and routes inward to the correct
present.

### 3. Present

The receiving being's present is where arriving boundary signal becomes part of
its operative reality.

## Motor

Motor beings are not universal.

They are purpose-bound beings created when outward world-facing expression is
required.

## Short Framing

The main sensory being is seeded on the genome path.

Peripheral input and motor beings are created later when purpose requires them.
