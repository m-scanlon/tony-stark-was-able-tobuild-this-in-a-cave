# v.04 notes

- `AcceptSignal` in metaxu needs to be renamed to `AcceptLanguage` — signals are now typed by language, not generic.
- The runtime is O(1). You're only paying for thought. — Map lookups, string parsing, struct copies. No graph traversal, no vector search, no embedding pipeline, no orchestration layer. A relation comes in, the world looks up the target in a map, builds the present, fires the medium. The only cost is inference. As models get faster and cheaper, Skyra gets faster and cheaper for free. There is nothing to optimize. Every dollar spent is a dollar spent on cognition, not plumbing.
- Blog post for July: "In the age of AI, the hardest skill is sitting still." — AI makes code free. Anyone can generate code. Code is no longer the bottleneck. Thinking is the bottleneck. The person who sits with the ontology until it's true is slower to start and untouchable once they move. Skyra spent two months not writing code — wrestling with the shape, compressing twelve architectures, throwing away everything that wasn't true enough. The implementation became trivial because the foundation was right. Thirty-minute features that nobody else has, on 674 lines, because the ontology was earned. Move fast and break things is dead. Sit still and find the shape. Let the AI write the code after. The ontology is the hard part. The code is notation.
